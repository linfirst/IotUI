# IotUI
这是一个通过socket连接的的Iot应用，还有很多地方需要优化。
如果有需要的话可以使用里面的UI。

折线图参考：
[TemperatureView](https://github.com/lyx19970504/TemperatureView)

1.增加可以动态绘制折线图功能
2.修改了连接处出现不重合的现象的代码

加减按钮参考：
[Android——自定义View之购物车的加减](https://blog.csdn.net/xy8199/article/details/78421658?utm_source=app)

仅对按钮进行了微调
1.增加了按钮背景
2.根据需要增加了单位
